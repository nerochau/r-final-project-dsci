# R Final Project – Is the National Exam a Good Measure of Student Success in Cambodia?

## Project Description

This project showcases applied statistical analysis using **R**, including data cleaning, exploratory analysis, visualization, and interpretation of results. The analysis was completed as a final project for a data science course.

## Key Skills Demonstrated

* Data wrangling and cleaning
* Statistical analysis
* Data visualization
* Reproducible research with R Markdown / Quarto

## File Overview

* `CHAU_final-proj.Rmd` – Main analysis source (R Markdown)
* `CHAU_final-proj.qmd` – Quarto version of the report
* `CHAU_final-proj.html` – Final rendered report
* `data/` – Input data
* `project.Rproj` – RStudio project configuration

## How to Reproduce

```text
Open project.Rproj → Open .Rmd or .qmd → Render / Knit
```

The complete analysis, figures, and interpretations can be viewed in the rendered HTML report:
